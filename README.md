# Alice

Distributed bot backend for Discord.

## Configuration

```Bash
BOT_TOKEN="maybe your bot token?"
# This MUST be the same between frontend and backend!
EVENT_QUEUE="memes"
# This MUST be the same between backend and cache!
CACHE_QUEUE="memes2"
SENTRY_DSN="your dsn here"
# Default "dev"
ENV_NAME="prod"
```
